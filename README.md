# Els

An experimental try for `ls` command. Expired from [EXA][01] which is written
in [Rust][02].

It is work-in-progress at the moment!

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'els'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install els
